Lab 4 - Pseudocoding and Problem Solving

Challenge

The goal of this lab was to get practice thinking using pseudocoding, serving as an intermediary step before just jumping straight into coding directly.

Problems

While we didn't run into any problems completing the tasks of the lab, the biggest problem we had was finding the time to complete everything.

Reflection

This lab shows how the idea and way of thinking of coding can be applied to many other tasks within our lives, as we can see with the first task of the lab. It's interesting to think about how learning to code can give us other skills beyond just the act of coding itself, and it's something that we can apply to our other interests.
